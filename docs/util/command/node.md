# Node 工具

## npm-check package升级
```bash
npm-check -u
```

## nvm 选择node版本
```bash
nvm ls                ## 查看已安装的所有node版本
nvm install node      ## 安装最新版本
nvm install version   ## 安装指定版本
nvm use version       ## 使用指定版本
nvm current           ## 查看当前版本
```

## nrm 选择仓库
```bash
nrm ls
nrm use taibao
```
